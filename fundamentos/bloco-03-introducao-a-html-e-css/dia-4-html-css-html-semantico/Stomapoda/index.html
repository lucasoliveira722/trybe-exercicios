Fatos sobre o Stomatopoda

Informações Gerais

O Stomatopoda (Odontodactylus scyllarus) é, sem sombra de dúvidas, um dos animais mais interessantes que existem no planeta.

Reino Filo Subfilo Classe Subclasse Ordem
Animalia Arthropoda Crustacea Malacostraca Hoplocarida Stomatopoda

Fato interessante 1 - Cones receptores de cores

Fato interessante 2 - Tiro de Garrinha

Fato interessante 3 - Possíveis de serem encontrados no litoral brasileiro

Vendo esses fatos, você deve estar pensando "mas esse bicho deve ser raro, e só deve existir em lugares como a Austrália ou a ilha de Madagascar", não é mesmo? Mas é aí que você se engana, caro leitor.
